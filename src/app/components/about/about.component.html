The application is divided into three sections:
  • Home
  • Statistics
  • About

Home

The Home page contains a button with the text 'New Game', when you click on it, a modal window will appear, where you will be prompted to select game settings, namely:

  • Username (from 3 to 10 symbols). Latin characters and numbers are allowed. The field is required.
  • Game area size (the number of cells in width and height) - two drop-downs. Possible width of the game area - 10 or 15 cells. When choosing 10 cells, the possible height of the game area is 5 or 6 cells. When choosing 15 cells, the possible height is 6, 7 or 8 cells. The default size of the game area is 10 x 5.
  • Game speed. Possible options are Low, Medium or High. The default speed is low, at a higher speed, the movement of ships, the shot and the movement of the sight are faster.
  • The maximum game time is set automatically when the speed is selected. At low speed - 60 seconds, at medium speed - 45 seconds, at high speed - 30 seconds. The default time is 60 seconds.

You can also reject the modal window with the 'Cancel' button, or start the game with the 'Start' button after the necessary settings. Please note, that you will not be able to launch the game if all settings are not selected.

After starting the game, a game area will appear, above which you can see: your name, a countdown timer, ammo and points counters, the selected speed and the pause button, when pressed, the game is paused, you can also pause the game by pressing the P key on the keyboard. You can resume the game at any time with the 'Continue' button or start a new game with different settings. The old game can be resumed in the Statistics page.

Game rules

The game area is divided into 3 zones: the skyline area, the sea area and the scope of the sight movement.

Ships float to the skyline area in random order, from either side. There are two types of ships: big and small, they move at a constant speed, depending on the selected game speed.

The sight is located at the bottom of the game area and is moved left and right with the <- and -> keys (arrows on the keyboard). A shot is fired with the Space button, it passes through the cells in the sea area and hits the ship if at the moment of activation of the upper cell, any part of the ship is above it. One shot can hit several ships. The next shot can be fired only if the bottom row does not contain active cells. The total number of points is 10. After hitting the target, the ship changes color to red, stops moving and disappears after a second.

    The total number of points is calculated according to the principle:
  • +150 for hitting a big ship
  • +300 for hitting a small ship
  • -50 for a miss

The score can not be lower than 0.

The game ends if all shots are used, or when the maximum time is over. After completing the game, the 'New Game' button will appear.

Statistics

    The Statistics page contains a table with the following data:
  • Username that was entered when the game was launched.
  • Game status (completed or paused). If the game is paused, then when you press the 'Paused' button you will be redirected to the game and when you press the P key or the Pause button, the game will continue.
  • The number of points is calculated based on the downed ships count and misses.
  • Game duration.
  • Used shots count (maximum 10).
  • Downed ships count (big and small).
  • Start date.
  • End date (if the game is paused, the field is empty).
  • Game area size, selected at the start of the game.
  • Game speed (low / medium / high), also selected when the game was launched.
    By default, the table is sorted:
  • by status (completed games are displayed first).
  • by points count (descending).
  • by username (in alphabetical order).
